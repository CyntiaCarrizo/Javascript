Escritura del codigo

Los identificadores (nombre que le damos a una variable, a una funcion, a una clase, etc) deben comenzar con:

  • Una letra
  • Un signo de dolar $
  • Un guión bajo _
  • Nunca con números o caracteres especiales.

Usa snake_case en:

  • Nombre de archivos: mi_archivo_javascript.js;

Usa UPPER_CASE en:

  • Constantes(variables que no cambian): const UNA_CONSTANTE = "Soy una constante",
  • PI = 3.141592653589793;

Usa UpperCamelCase (letra inicial en mayuscula) en:

  • Clases

Usa lowerCamelCase (si tu variable o funciones contiene mas de 2 palabras, a partir de la segunda palabra la letra inicial va con mayuscula) en:

  • Objetos
  • Primitivos
  • Funciones
  • Instancias

Se utiliza el acento invertido (alt 96), el signo dolar y entre llaves el nombre de la variable. Los template String permiten los saltos de lineas no asi las cadenas de textos.
